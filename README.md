
# 🚀 Portfólio - Bernardo Almeida

📌 Sobre o Projeto

Este é meu portfólio pessoal de desenvolvedor Fullstack, criado para apresentar minhas habilidades, projetos e tecnologias que utilizo no dia a dia.

O objetivo do projeto é servir como uma vitrine profissional, permitindo que recrutadores e desenvolvedores conheçam melhor meu trabalho, experiências e forma de desenvolver interfaces modernas.
